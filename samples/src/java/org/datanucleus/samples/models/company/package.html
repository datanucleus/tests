This package provides a few simple classes based around an organisation or company. They provide inheritance, composition, and relationships.
  • Person - base class for all people
  • Employee - extends Person, representing the people in a company. Many-One (bi) relation with Manager. One-One (uni) relation with Account.
  • Manager - a manager of Employees, extending Employee. One-Many (bi) relation with Department. One-Many (bi) relation with Employee.
  • Department - an organisational unit of a company. Many-One (bi) relation with Manager
  • Account - a login account for an Employee.
  • Office - an office. Set of departments.
