Series of classes that could be used for storing embedded within each other.
  • FilmLibrary has a Map of Film (1-N uni) and these could be stored embedded.
  • Computer has 2 ComputerCards (1-1 bi) and these could be stored embedded
  • Processor has a List of Jobs (1-N uni) and these could be stored embedded
  • Network has a Set of Devices (1-N bi) and these could be stored embedded
  • MusicPlayer has a Batterys (1-1 uni) and this could be stored embedded
  • DigitalCamera has a Memorys (1-1 uni) and in turn has a Chip and both could be embedded (nested).
