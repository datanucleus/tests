Package providing examples of all possible Map relations between container and key/value classes.
  • MapHoldern class with all types of Map fields demonstrating all possible Map combos
  • MapKeyItem - PC type of key when using join table
  • MapValueItem - PC type of value when using join table
  • MapFKKeyItem - PC type of key when the value is part of the key
  • MapFKValueItem - PC type of value when the key is part of the value
